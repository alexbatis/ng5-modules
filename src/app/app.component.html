Multiple Module Angular Single Page Application

This application was built with 5 modules:

  • AppModule - parent module which is rendered
  • HomeModule - holds content of home page
  • ContentModule - holds content of content page
  • AboutModule - holds content of about page
  • SharedModule - declares and exports shared components across modules

Each Module has a 'RootModule' which is bootstrapped to the parent module. This root module can hold the HTML/CSS/TS for the content of that module.

In the SharedModule, the PageHeaderComponent is defined and used across modules. In Angular 5, a component can only be declared in one module. Using a shared module allows for a component to be used in any module that has the shared module as an import.

This example also uses a shared service, which is delcared in the providers section of the AppModule and used in each other module.
